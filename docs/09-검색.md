# 검색

## 개요

전체 단어장에 걸쳐 단어를 검색하는 글로벌 검색 기능이다. 홈 대시보드와 단어장 목록 탭의 검색 아이콘을 통해 전체화면 모달로 진입한다. 단어의 영단어(term), 한국어 뜻(meaningKr), 영어 정의(definition), 태그(tags)를 대상으로 검색하며, 단어장 필터와 별표 필터를 조합할 수 있다.

## 관련 파일

- `app/search-modal.tsx` — 검색 전체화면 모달

## 주요 개념

| 개념 | 설명 |
|------|------|
| 글로벌 검색 | `lists` 상태의 모든 단어를 대상으로 검색 (숨겨진 단어장 포함) |
| 검색 대상 필드 | `term`, `meaningKr`, `definition`, `tags[]` (대소문자 무관) |
| Top 5 태그 | 전체 단어에서 빈도수 상위 5개 태그를 퀵필터 칩으로 표시 |
| `starredOnly` | 별표 단어만 필터링. 검색어 없이 단독으로도 동작 |

## 동작 시나리오

### 시나리오 1: 단어 검색

1. 홈 또는 단어장 탭의 검색 아이콘을 탭한다.
2. `search-modal`이 슬라이드 업 애니메이션으로 열린다.
3. 상단에 **Top 5 태그** 칩이 표시된다.
   - 태그 칩 탭 → 해당 태그가 검색창에 입력되어 즉시 검색 실행
4. 검색창에 텍스트를 입력한다.

> **검색 로직** (`useMemo`):
> ```
> 전체 lists의 모든 단어를 [{ word, listName, listId }] 배열로 수집
> ↓
> selectedListId 필터 적용 (특정 단어장 선택 시)
> ↓
> starredOnly 필터 적용
> ↓
> trimmed 검색어로 아래 필드 OR 조건 검색:
>   - word.term.toLowerCase().includes(trimmed)
>   - word.meaningKr.toLowerCase().includes(trimmed)
>   - word.definition.toLowerCase().includes(trimmed)
>   - word.tags.some(t => t.toLowerCase().includes(trimmed))
> ```

5. 결과 카드에 단어명, 한국어 뜻, 소속 단어장 이름이 표시된다.
6. 결과 카드를 탭하면 `add-word` 모달이 `mode: 'read'`로 열려 단어 상세 정보를 확인할 수 있다.

---

### 시나리오 2: 단어장 필터 적용

1. 검색 화면에서 단어장 선택 드롭다운(또는 칩)을 탭한다.
2. 특정 단어장을 선택하면 `selectedListId`가 설정된다.
3. 검색 결과가 해당 단어장 내 단어로만 필터링된다.
4. "전체" 선택 시 `selectedListId = null`로 전체 검색 복원.

---

### 시나리오 3: 별표 단어만 보기

1. 별표 토글 아이콘을 탭한다.
2. `starredOnly = true`로 설정된다.
3. 검색어 없이도 별표 단어 전체가 표시된다 (검색어가 있으면 AND 조건으로 추가 필터).

---

### 시나리오 4: 태그 퀵 검색

1. 화면 상단의 Top 5 태그 칩 중 하나를 탭한다.
2. `handleTagClick(tag)` → `setQuery(tag)` + 햅틱 피드백
3. 해당 태그를 포함하는 단어들이 즉시 검색 결과로 표시된다.

> **Top 5 태그 계산 로직**:
> ```javascript
> const tagCount = {};
> allData.forEach(({ word }) => {
>   word.tags.forEach(t => { tagCount[t] = (tagCount[t] || 0) + 1; });
> });
> return Object.entries(tagCount)
>   .sort((a, b) => b[1] - a[1])
>   .slice(0, 5)
>   .map(entry => entry[0]);
> ```

## 예외 / 엣지 케이스

- **빈 검색어 + `starredOnly = false`**: 결과 없음 (빈 배열 반환) — 빈 검색어로 전체 단어 나열 방지
- **빈 검색어 + `starredOnly = true`**: 별표 단어 전체 표시
- **숨겨진 단어장**: `lists` 전체를 대상으로 검색하므로 `isVisible = false`인 단어장의 단어도 검색됨
- **태그 없는 단어**: Top 5 태그 계산 시 태그 배열이 비어있으면 건너뜀 (에러 없음)
- **대소문자 비구분**: 검색어와 모든 필드를 `.toLowerCase()`로 통일하여 비교
