# 설정

## 개요

설정 탭에서 앱의 다양한 동작 방식을 커스터마이징할 수 있다. 설정은 6개 그룹으로 분리되어 각각 AsyncStorage에 독립적으로 저장된다. 테마(다크모드), 앱 언어, 계정 관리, 학습 설정, 입력 설정 등을 포함한다.

## 관련 파일

- `app/(tabs)/settings.tsx` — 설정 화면 UI
- `contexts/SettingsContext.tsx` — 6개 설정 그룹 상태 관리 및 영속화
- `contexts/ThemeContext.tsx` — 다크/라이트 모드 관리
- `contexts/LocaleContext.tsx` — 앱 언어 관리
- `i18n/index.ts` — 지원 언어 목록(`UI_LOCALES`)

## 주요 개념

| AsyncStorage 키 | 설정 그룹 | 내용 |
|----------------|----------|------|
| `@soksok_user_input_settings` | InputSettings | 단어 추가 폼 관련 설정 |
| `@soksok_user_study_settings` | StudySettings | 학습 배치 크기 |
| `@soksok_user_autoplay_settings` | AutoPlaySettings | 오토플레이 관련 설정 |
| `@soksok_profile_settings` | ProfileSettings | 닉네임, 시작 탭 |
| `@soksok_custom_study_settings` | CustomStudySettings | 맞춤학습 설정 |
| `@soksok_dashboard_filter` | DashboardFilterMode | 홈 대시보드 필터 |
| `@soksok_theme` | ThemeContext | 라이트/다크 모드 |
| `@soksok_locale` | LocaleContext | 앱 언어(ko/en) |

## 설정 그룹별 항목

### 1. InputSettings (단어 입력 설정)
| 항목 | 기본값 | 설명 |
|------|--------|------|
| `showDefinition` | false | 영어 정의 필드 표시 여부 |
| `showPos` | false | 품사(POS) 필드 표시 여부 |
| `showExample` | true | 예문 필드 표시 여부 |
| `showTags` | true | 태그 필드 표시 여부 |
| `showPhonetic` | true | 발음기호 필드 표시 여부 |
| `addWordMode` | `'popup'` | 단어 추가 방식: 팝업 or 전체화면 |
| `fieldOrder` | `['term','meaningKr','pos',...]` | 입력 필드 순서 (드래그로 재배치) |
| `sourceLang` | `'en'` | 원본 언어 |
| `targetLang` | `'ko'` | 번역 목표 언어 |
| `enableAutocomplete` | true | 자동완성(네이버/Datamuse) 활성화 |

### 2. StudySettings (학습 설정)
| 항목 | 기본값 | 설명 |
|------|--------|------|
| `studyBatchSize` | `'all'` | 플래시카드/퀴즈 배치 크기 (5/10/20/50/전체) |
| `sentenceBatchSize` | `'all'` | 예문학습 배치 크기 |

### 3. AutoPlaySettings (오토플레이 설정)
| 항목 | 기본값 | 설명 |
|------|--------|------|
| `filter` | `'all'` | 단어 필터 (전체/학습중/암기완료) |
| `isStarred` | false | 별표 단어만 표시 |
| `showTerm` | true | 단어 표시 |
| `showMeaning` | true | 뜻 표시 |
| `showPos` | true | 품사 표시 |
| `showExample` | true | 예문 표시 |
| `showExampleKr` | true | 예문 번역 표시 |
| `autoPlaySound` | true | TTS 자동 재생 |
| `delay` | `'2s'` | 자동 넘기기 간격 |

### 4. ProfileSettings (프로필 설정)
| 항목 | 기본값 | 설명 |
|------|--------|------|
| `nickname` | `''` | 커뮤니티 공유 시 표시될 이름 |
| `startupTab` | `'index'` | 앱 시작 시 열리는 탭 (홈/단어장/큐레이션) |

## 동작 시나리오

### 시나리오 1: 앱 언어 변경

1. 설정 → "언어" 항목을 탭한다.
2. `ModalPicker`로 지원 언어 목록 표시 (현재: 한국어, English).
3. 언어를 선택하면 `setLocale(code)` 호출:
   - `i18n.changeLanguage(code)` 즉시 실행 → 모든 `t()` 번역 함수가 새 언어로 전환
   - `AsyncStorage.setItem('@soksok_locale', code)` 저장
4. 설정 화면 포함 전체 UI가 즉시 선택된 언어로 변경된다.

> **내부 로직**: `LocaleContext`의 `setLocale`이 i18next의 `changeLanguage`를 호출하므로 화면 리로드 없이 즉시 전환.

---

### 시나리오 2: 다크 모드 전환

1. 설정 → "다크 모드" 스위치를 토글한다.
2. `toggleTheme()` 호출:
   - `mode`를 `'light'` ↔ `'dark'`로 전환
   - `AsyncStorage.setItem('@soksok_theme', next)` 저장
3. `ThemeContext`의 `colors`가 즉시 `Colors.dark` 또는 `Colors.light`로 바뀌어 전체 UI 색상 전환.

> **내부 로직**: 앱 초기 실행 시 AsyncStorage에 저장된 값을 읽어 복원. 저장값 없으면 기기 시스템 색상 설정(`useColorScheme()`)을 기본값으로 사용.

---

### 시나리오 3: 시작 탭 설정

1. 설정 → "시작 화면" 항목을 탭한다.
2. `ModalPicker`로 3개 탭 선택: 홈 / 단어장 / 큐레이션.
3. `updateProfileSettings({ startupTab: 'vocab-lists' })` 저장.
4. 다음 앱 실행 시 탭 레이아웃(`app/(tabs)/_layout.tsx`)에서 `profileSettings.startupTab` 을 읽어 해당 탭으로 초기 진입.

---

### 시나리오 4: 닉네임 설정

1. 설정 → 계정 섹션의 닉네임 영역을 탭한다.
2. `DialogModal`이 열리고 현재 닉네임이 입력창에 표시된다.
3. 새 닉네임 입력 후 "저장"을 누른다.
4. `updateProfileSettings({ nickname: input.trim() })` 저장.
5. 단어장 공유 시 `profileSettings.nickname`이 `creatorName`으로 사용된다. 닉네임이 비어있으면 Google 사용자 `displayName` 또는 'Anonymous' 폴백.

---

### 시나리오 5: 앱 시작 시 설정 로드

1. 앱 시작 시 `SettingsProvider`의 `loadSettings()`가 AsyncStorage에서 6개 키를 **병렬로** 읽는다.
2. 각 저장값을 기본값(`DEFAULT_*`)에 `Object.assign` 형태로 병합한다 (새 필드 추가 시 기본값 보장).
3. `isLoading`이 `false`가 되면 설정이 준비된 것으로 간주.

> **필드 순서 마이그레이션**: `fieldOrder` 배열에 새 필드(예: `'pos'`, `'phonetic'`)가 없는 경우 올바른 위치에 자동 삽입하는 마이그레이션 로직이 있어 앱 업데이트 후에도 기존 사용자 설정 유지.

## 예외 / 엣지 케이스

- **AsyncStorage 로드 실패**: `try/catch`로 무시하고 기본값으로 폴백
- **게스트 사용자 계정 섹션**: Google 프로필 이미지/이름 대신 "게스트 모드" 텍스트와 아이콘 표시. 닉네임, 클라우드 동기화 배지 없음
- **언어 변경 영속성**: 앱 재시작 후에도 선택한 언어 유지 (AsyncStorage)
- **시스템 다크모드**: 앱 내에서 따로 설정하지 않으면 기기 시스템 설정을 따름. 한 번 수동 설정하면 시스템 설정과 무관하게 고정
